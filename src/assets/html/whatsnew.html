What is new in ThinkSystem Intelligent Monitoring OneCollect 1.9

ThinkSystem Intelligent Monitoring OneCollect introduces support for the new features:
  • Subnet discovery and Simple Network Management Protocol (SNMP) community string-based discovery
  • Credential Management to manage default credentials for devices
  • Export auto-discovery results into Excel spreadsheet
The following issue has been addressed in ThinkSystem Intelligent Monitoring OneCollect 1.9:
  • HBA details parsing in AIX and Solaris
  • iSCSI parsing updates for ESXi and vCenter
This release includes important enhancement:
  • Auto-discovery results and performance
  • The graphical user interface (GUI) has been revamped to view and identify different types of jobs
  • Improved the "Scheduled Jobs" page to display next schedule and status of the previous schedule
